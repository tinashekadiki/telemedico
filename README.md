##Telemedico

Backend for Telemedico  
**Tinashe Kadiki's 4th year dissertation project**
##Usage

- Install Dependencies  
`npm install`

- Run Server  
`npm start`  
 OR  
 Install `nodemon` if you do not have it already installed and run the following command  
 `npm run dev`
